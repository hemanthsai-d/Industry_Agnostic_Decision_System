Observability

Monitoring & Dashboards

Real-time visibility into system health, performance, and AI quality. Prometheus collects data, Grafana visualizes it, and Alertmanager pages the team when something needs attention.

📊

Prometheus

The data collector. Every 15 seconds, it pulls numbers from the API and model server — latency, error rates, decision counts, and more. Think of it as the system's heartbeat monitor.

📈

Grafana

The visual dashboard. Turns those raw numbers into beautiful, real-time graphs and charts. Engineers and managers can see at a glance if everything is healthy.

🔔

Alertmanager

The alarm system. When metrics cross danger thresholds, it pages the right person immediately. Critical issues: page in 5 minutes. Warnings: notify the team channel.

Prometheus

25+ Monitored Metrics

Every important aspect of the system is tracked with Prometheus metrics. Here's what we monitor and why.

MetricTypeWhat It TracksWhy It Matters
assist_http_requests_total Counter Every HTTP request by method, path, status code Know exactly how many requests per endpoint and if errors are increasing
assist_http_request_duration_seconds Histogram How long each request takes (p50, p95, p99) Catch slowdowns before customers notice
assist_http_request_errors_total Counter Exceptions and 5xx errors by type Spot bugs and failures as they happen
assist_inflight_requests Gauge Current number of requests being processed Detect overload and capacity issues in real-time
assist_rate_limit_exceeded_total Counter Rate limit rejections by scope (tenant/user) Track abuse attempts and ensure fair usage
MetricTypeWhat It TracksWhy It Matters
assist_decisions_total Counter Decision outcomes: recommend, abstain, or escalate Track the balance of AI confidence vs human escalations
assist_decision_confidence Histogram Distribution of confidence scores across all decisions Detect if the model is becoming overconfident or too cautious
assist_handoffs_total Counter How many cases escalated to human agents Monitor escalation rates and ensure AI doesn't overwhelm human teams
assist_decision_cache_hits_total Counter Idempotent request cache hits Prevent duplicate processing of the same request
assist_shadow_predictions_total Counter Challenger model shadow predictions logged Track experimental model performance without affecting production
MetricTypeWhat It TracksWhy It Matters
assist_pipeline_stage_duration_seconds Histogram Time spent in each pipeline stage (PII, injection, retrieval, routing, generation) Pinpoint exactly which stage is slowing down when latency spikes
assist_pipeline_total_duration_seconds Histogram Total end-to-end pipeline wall time The definitive measure of user-facing latency
assist_injection_detections_total Counter Prompt injection attempts (blocked or filtered) Security monitoring — detect attack patterns
assist_issue_text_token_count Histogram Input length distribution Detect unusual input patterns that could indicate drift or abuse
assist_model_guardrail_fallback_total Counter Times the system fell back from ML model to heuristic rules Critical safety metric — high fallback rate = model instability
MetricTypeWhat It TracksWhy It Matters
assist_rag_faithfulness Histogram How well AI responses match the source evidence Low faithfulness = the AI is making things up (hallucinating)
assist_rag_hallucination_ratio Histogram Percentage of responses with unsupported claims Alert if hallucinations exceed 35% — serious quality issue
assist_rag_citation_coverage Histogram How often responses include proper evidence citations Ensures accountability — every claim should link back to a source
assist_retrieval_evidence_score Histogram Quality scores of retrieved knowledge base chunks Low scores mean the knowledge base needs updating or the question is new
MetricTypeWhat It TracksWhy It Matters
assist_request_cost_dollars Histogram USD cost per request (by generation backend) Track infrastructure costs and optimize spending
assist_routing_accuracy Gauge Daily classification accuracy (from evaluation) The headline accuracy number — updated daily via evaluation pipeline
assist_escalation_recall Gauge How well the system catches cases that need escalation Missing escalations = customers getting wrong answers
assist_calibration_ece Gauge Expected Calibration Error — how honest the confidence scores are Low ECE = when the system says "80% sure", it's right ~80% of the time
assist_abstain_rate Gauge Percentage of requests where the system defers to humans Too high = wasted automation; too low = risky over-confidence

Prometheus Scrape Configuration

How Prometheus connects to the services to collect metrics.
